HTML, CSS, and JavaScript: Building the Web

Web Development Certification

This certification from linkedin Learning focuses on the foundational technologies that power the modern web. Through this course, I developed a deeper understanding of how websites are structured, styled, and made interactive using the core languages of the internet. The course explored the relationship between HTML for structure, CSS for design and layout, and JavaScript for functionality and dynamic user interaction.

Figure 1: Certificate of Completion – HTML, CSS, and JavaScript: Building the Web

Core Competencies Developed:

• HTML Structure – Creating semantic webpage layouts using modern HTML standards.
• CSS Styling – Designing responsive layouts and visually structured interfaces.
• JavaScript Fundamentals – Implementing interactive elements and dynamic behaviors within websites.
• Web Development Workflow – Understanding how the three core technologies work together to create modern websites.
